Making A Handwritten Digit Recognizer Using MNIST Dataset

(Which Works On Images Clicked By You)

Not another blog post which will show you how to create a model which just tells you "IMAGE CLASSIFIED CORRECTLY". We'll use Pillow and OpenCV python libraries to preprocess the images and tensorflow to train our model.
